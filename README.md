# 📉 Customer Churn Analysis

## 📌 Overview
This project analyzes customer churn to identify patterns and reduce customer loss.

## 🎯 Objectives
- Identify churn rate
- Analyze churn drivers
- Improve retention strategies

## 🛠 Tools Used
- Python (Pandas)
- Excel
- Power BI

## 📂 Dataset
- 7,000+ customer records
- Fields: tenure, MonthlyCharges, Contract, PaymentMethod, Churn

## 🔄 Workflow
1. Data Cleaning (handle missing values)
2. EDA (churn patterns)
3. Advanced Analysis (risk segmentation)
4. Dashboard creation

## 📊 Key Insights
- 26% customers churned
- New customers churn more
- Month-to-month contracts have highest churn
- Electronic check users churn more

## 📈 Dashboard Features
- Churn KPIs
- Churn by Contract
- Churn by Tenure
- Payment Method Analysis

## 📌 Conclusion
The analysis helps reduce churn through targeted retention strategies.
